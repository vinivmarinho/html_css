PRINCIPAIS FORMATAÇÕES:

Negrito/destaque

Nessa frase, a parte a seguir está em negrito, utilizando a tag <b> (Não semântica)

Nessa frase, a parte a seguir está em destaque, utilizando a tag <STRONG> (Semântica)

ITÁLICO/ENFÂSE

Nessa frase, a parte a seguir está em itálico, utilizando a tag <i> (Não semântica)

Nessa frase, a parte a seguir tem um termo com enfâse, utilizando a tag <em> (Semântica)

TEXTO MARCADO

Marcando o texto utilizando a tag <&mark>

TEXTO GRANDE E PEQUENO

No texto grande, a tag obsoleta que era usada seria o <big> (não semântica, por isso agora é obsoleta)

No texto pequeno,é utilizada a tag <small> (semântica e correta)

TEXTO DELETADO

Para deletar uma parte to texto, utilizar a tag <del>

TEXTO INSERIDO

Para inserir um texto, utilizar a tag <ins>

TEXTO SOBRESCRITO

Para inserir coisas do tipo x20+3, é usada a tag <sup>

TEXTO SUBSCRITO

Para inserir coisas do tipo H2O, é utilizada a tag <sub>
